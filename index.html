HTML

<div>
La balise div est un block pouvant contenir tout type de contenu : autres balises, textes, images, etc.
<div>
     <p>Bonjour</p>
</div>
<div>

</div>
<p>
La balise p a été créé pour représenter des paragraphes, elle a naturellement des marges en haut et en bas.
<p>Je suis un paragraphe de texte</p>
<p>Je suis un paragraphe de texte</p>
<a>
La balise a est utilisée pour les hyperliens. Elle prend au minimum un attribut href.
<p>Voici un <a href="/chemin/vers/ma/page">lien</a>.</p>
<p>Voici un <a href="/chemin/vers/ma/page">lien</a>.</p>
<img>
Balise utilisée pour déclarer une image. Elle prend au minimum l'attribut src , et il est de bon goût d'utiliser l'attribut alt pour les lecteurs d'écrans. Elle est auto-fermante.
<a src="/chemin/vers/mon/image" alt="Description de l'image" />
<a src="/chemin/vers/mon/image" alt="Description de l'image" />
<ul> / <ol>
Ce sont les balises pour créer des listes. <ul> crée une liste non ordonnée (liste à puces), alors que <ol> crée une liste ordonnée (liste numérotée). Elles contiennent des balises <li> .
<ol>
     <li>Préchauffer le four</li>
     <li>Préparer les cookies</li>
     <li>Enfourner la plaque de cookies</li>
</ol>
  1. Préchauffer le four
  2. Préparer les cookies
  3. Enfourner la plaque de cookies
<ul>
     <li>Pain</li>
     <li>Beurre demi-sel</li>
     <li>Lait</li>
</ul>
  • Pain
  • Beurre demi-sel
  • Lait
<button>
Crée un bouton (utilisé pour les actions utilisateurs)
<button>Clique-moi</button>
<input>
Cette balise crée un champ de texte (mais aussi nombre, mot de passe, zone de texte). On lui donne un type.
<input type="text" placeholder="Entrez du texte">
