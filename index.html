Feuille récap HTML/CSS/JS

HTML

<div>
La balise div est un block pouvant contenir tout type de contenu : autres balises, textes, images, etc.
<div>
     <p>Bonjour</p>
</div>

Bonjour

<p>
La balise p a été créé pour représenter des paragraphes, elle a naturellement des marges en haut et en bas.
<p>Je suis un paragraphe de texte</p>

Je suis un paragraphe de texte

<a>
La balise a est utilisée pour les hyperliens. Elle prend au minimum un attribut href.
<p>Voici un <a href="/chemin/vers/ma/page">lien</a>.</p>

Voici un lien.

<img>
Balise utilisée pour déclarer une image. Elle prend au minimum l'attribut src , et il est de bon goût d'utiliser l'attribut alt pour les lecteurs d'écrans. Elle est auto-fermante.
<img src="/chemin/vers/mon/image" alt="Description de l'image" />
chemin

Comme vu au-dessus pour la balise img, lorsqu'on veut utiliser un fichier, on doit le lier en indiquant son chemin.

On peut utiliser le chemin absolu, à savoir le chemin depuis la racine (du domaine ou de l'ordinateur par exemple). Il commence par le symbole /

On peut aussi utiliser le chemin relatif, à savoir le chemin relatif au fichier courant. Dans ce cas on utilise le ./ pour indiquer que l'on démarre du répertoire courant (aussi appelé dossier) contenant le fichier dans lequel on code. Mais on peut aussi utiliser le ../ pour indiquer qu'on souhaite démarrer dans le répertoire parent (celui qui contient le répertoire courant).


<img src="./chemin-relatif" alt="Description de l'image" />

<img src="../autre/chemin-relatif" alt="Description de l'image" />

<img src="/chemin-absolu" alt="Description de l'image" />
<ul> / <ol>

<ol>
     <li>Préchauffer le four</li>
     <li>Préparer les cookies</li>
     <li>Enfourner la plaque de cookies</li>
</ol>
  1. Préchauffer le four
  2. Préparer les cookies
  3. Enfourner la plaque de cookies
 
<ul>
     <li>Pain</li>
     <li>Beurre demi-sel</li>
     <li>Lait</li>
</ul>
  • Pain
  • Beurre demi-sel
  • Lait

CSS

Les sélecteurs CSS

Pour sélectionner les balises à styliser avec du CSS, on utilise des sélecteurs CSS.

balise
On peut sélectionner les balises via leur nom. Cela sélectionnera toutes les balises de ce type.
<div >Un bloc informatif</div>

div {
  color: red;
}
Un bloc informatif
.class
On peut ajouter l'attribut class="" à la balise, voire à plusieurs balises de même type ou de types différents.
<div class="infos">Des infos</div>

.infos {
  color: red;
}
Des infos
#id
On peut ajouter l'attribut id="" à la balise. Il doit être unique.
<button id="bouton-sauver">Enregistrer les infos</button>

#bouton-sauver {
  color: grey;
}
balise .class
Il est possible de chaîner les sélecteurs CSS pour préciser la ou les balises visée(s).
<p class="text">Un chouette texte.</p>
<div>
    <p class="text">Un texte très sympa.</p>
</div>

div .text {
  color: red;
}

Un chouette texte.

Un texte très sympa.

JS

variable
Une variable contient une valeur.

Il existe différents type: chaîne de caractère ( "coucou" ou 'coucou' ), nombre ( 5 ), tableau ( ["coucou", 5, "avion"] ), objet ( { nom: "Blier", prenom: "Juliane", age: 28 } ).

Pour utiliser une variable, on la déclare, puis on peut lui assigner une valeur. Ou alors on la déclare en lui assignant directement une valeur.


Pour accéder à une valeur dans un tableau on utilise son index entre [ ]. L'index commence toujours à 0.
Pour accéder à une valeur dans un objet, on utilise le nom de l'objet et la clé de la valeur sous la forme : objet.cle
let prenom;
prenom = "Juliane";

let age = 28;

let liste = ["pain", "lait", "oeufs"];

let identite = {
  prenom: "Juliane",
  age: 28,
  signes: ["lunettes"]
}

console.log( prenom ) //Juliane
console.log( liste[0] ) //pain
console.log( identite.age ) //28
fonction
Une fonction est un morceau de code que l'on crée pour réutiliser les instructions. Elle peut avoir arguments (aussi appelés paramètres) qu'on lui passe.

On doit la déclarer au préalable pour pouvoir l'utiliser. Ensuite on peut l'appeler en lui passant des arguments si elle en nécessite.

Elle peut retourner (renvoyer) une valeur grâce au mot-clé return, ou juste appliquer une série d'instructions.
function addition (arg1, arg2) {
     let resultat = arg1 + arg2;
     return resultat;
}

let total = addition(4, 6);
console.log( total ) //10
getElementById
On peut sélectionner une balise HTML en javascript, en sélectionnant son id grâce à la méthode (une sorte de fonction) getElementById.
Il faut alors indiquer dans les parenthèses l'id à sélectionner, et ce, entre " ".
<div id="mon-id">Une div que je veux sélectionner.</div>

let element = document.getElementById("mon-id")
.innerHTML

Le contenu de cette balise sera remplacée

let element = document.getElementById("mon-id")
element.innerHTML = "Bonjour"

Vue.js

component
Pour déclarer un composant, il faut:
  • l'importer
  • lui donner un nom
  • l'utiliser dans le composant parent
Dans la balise script

import MonComposant from './MonComposant.vue';

export default {
  name: "MonComposant"
}


Dans le composant parent

<mon-composant></mon-composant>
v-if
Le v-if permet d'afficher ou non la balise HTML sur laquelle il est placé.
On lui passe une donnée contenue dans la data() du composant. Celle-ci est évaluée à true (vrai) ou false (faux).
Dans le template HTML

<mon-composant v-if="afficher"></mon-composant>

Dans la balise script

export default {
  name: "MonComposant",
  data() {
    return {
      afficher: true
    }
}
data
On peut utiliser la donnée directement dans le template HTML grâce aux {{ }}
Il faut que la donnée soit déclarée dans la data de notre composant. Vue, fait directement le lien entre les deux.
Dans le template HTML

<div>{{ nomDuResto }}</div>

Dans la balise script

export default {
  name: "MonComposant",
  data(){
    return {
      nomDuResto: "Bistrot Régent"
    }
  }
}
v-on
Le v-on permet d'écouter un événement grâce à Vuejs.
On lui indique le type d'événement et la fonction à appeler. Lorsque l'événement est déclenché, il lance la fonction indiquée.
Dans le template HTML

<mon-composant v-on:click="ajouterTrois(6)"></mon-composant>

Dans la balise script

export default {
  name: "MonComposant",
  methods: {
    ajouterTrois(nombre) {
      return nombre + 3;
    }
  }
}
v-for
Le v-for est une boucle permettant d'afficher tous les éléments d'une liste
Pour chaque tour de boucle, on affichera l'élément html correspondant.
Dans le template HTML

<mon-composant>
  <p v-for="resto in liste">{{ resto }}</p>
</mon-composant>

Dans la balise script

export default {
  name: "MonComposant",
  data(){
    return {
      liste: ["Bistrot Régent", "Le Tranchoir", "La Belle verte"]
    }
  }
}

Bistrot Régent

Le Tranchoir

La Belle verte

props
On peut passer des informations à un composant via des props.
On crée un attribut nommé comme on le souhaite, et on lui indique le texte voulu.
Pour passer une donnée de manière dynamique, on ajoute v-bind: devant la props et on lui passe la bonne donnée.

Le v-bind sert à indiquer que l'attribut est dynamique.
Dans le template HTML

<mon-composant telephone="05 90 90 90 45" v-bind:nom="nomDuResto"></mon-composant>

Dans la balise script

export default {
  name: "MonComposant",
  props: ['telephone'],
  data() {
    return {
      nomDuResto: 'Bistrot Régent'
    }
  }
}
mounted
Si on veut agir au moment où le HTML est chargé, on utilise la fonction mounted().
On l'appelle et on indique le code que l'on veut exécuter à l'intérieur.
Dans le template HTML

<mon-composant></mon-composant>

Dans la balise script

export default {
  name: "MonComposant",
  mounted() {
    displayText()
  }
}
