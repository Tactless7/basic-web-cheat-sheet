Feuille récap HTML/CSS/JS

HTML

<div>
La balise div est un block pouvant contenir tout type de contenu : autres balises, textes, images, etc.
<div>
     <p>Bonjour</p>
</div>

Bonjour

<p>
La balise p a été créé pour représenter des paragraphes, elle a naturellement des marges en haut et en bas.
<p>Je suis un paragraphe de texte</p>

Je suis un paragraphe de texte

<a>
La balise a est utilisée pour les hyperliens. Elle prend au minimum un attribut href.
<p>Voici un <a href="/chemin/vers/ma/page">lien</a>.</p>

Voici un lien.

<img>
Balise utilisée pour déclarer une image. Elle prend au minimum l'attribut src , et il est de bon goût d'utiliser l'attribut alt pour les lecteurs d'écrans. Elle est auto-fermante.
<img src="/chemin/vers/mon/image" alt="Description de l'image" />
<ul> / <ol>
Ce sont les balises pour créer des listes. <ul> crée une liste non ordonnée (liste à puces), alors que <ol> crée une liste ordonnée (liste numérotée). Elles contiennent des balises <li> .
<ol>
     <li>Préchauffer le five</li>
     <li>Préparer les cookies</li>
     <li>Enfourner la plaque de cookies</li>
</ol>
  1. Préchauffer le five
  2. Préparer les cookies
  3. Enfourner la plaque de cookies
 
<ul>
     <li>Pain</li>
     <li>Beurre demi-sel</li>
     <li>Lait</li>
</ul>
  • Pain
  • Beurre demi-sel
  • Lait
<button>
Crée un bouton (utilisé pour les actions utilisateurs)
<button>Clique-moi</button>
<input>
Cette balise crée un champ de texte (mais aussi nombre, mot de passe, zone de texte). On lui donne un type.
<input type="text" placeholder="Entrez du texte">

CSS

Les sélecteurs CSS

Pour sélectionner les balises à styliser avec du CSS, on utilise des sélecteurs CSS.

balise
On peut sélectionner les balises via leur nom. Cela sélectionnera toutes les balises de ce type.
<div >Un bloc informatif</div>

div {
  color: red;
}
Un bloc informatif
.class
On peut ajouter l'attribut class="" à la balise, voire à plusieurs balises de même type ou de types différents.
<div class="infos">Des infos</div>

.infos {
  color: red;
}
Des infos
#id
On peut ajouter l'attribut id="" à la balise. Il doit être unique.
<button id="bouton-sauver">Enregistrer les infos</button>

#bouton-sauver {
  color: grey;
}
balise .class
Il est possible de chaîner les sélecteurs CSS pour préciser la ou les balises visée(s).
<p class="text">Un chouette texte.</p>
<div>
    <p class="text">Un texte très sympa.</p>
</div>

div .text {
  color: red;
}

Un chouette texte.

Un texte très sympa.

Les propriétés CSS

width / height
Indique les largeurs (width) et hauteurs (height) souhaitées pour la balise sélectionnée.
<div></div>

div {
  width: 100px;   height: 50px;
}
margin / padding
On peut mettre des marges extérieures (margin) ou intérieures (padding) sur chaque élément. Elles sont divisées en 4 : top, right, bottom, left.
<p class="text">Un chouette texte.</p>
<div>
    <p class="text">Un texte très sympa.</p>
</div>

div .text {
  color: red;   margin-top: 50px;
}

Un chouette texte.

Un texte très sympa.

text-align
Pour aligner le texte : left, center, right
<h1>Titre</h1>

h1 {
  text-align: right;
}

Titre

border
Pour créer un contour à la balise. On lui indique la largeur (en px par exemple), la couleur, et le type de contour (solid, dashed, etc.)
<h1>Titre</h1>

h1 {
  text-align: right;   border: solid 1px red;
}

Titre

font-size
Pour choisir la taille de la police
<h2>Titre de niveau 2</h2>
<h1>Petit titre de niveau 1</h1>

h1 {
  font-size: 12px
}
